## Notes

- Linux is not the authoritative host for macOS DMG packaging because the DMG target depends on Darwin native modules. Linux release verification stops at `corepack pnpm build:electron`; `.app`/`.dmg` artifacts are produced on macOS.
- `dmg-license@1.0.11` is pinned as a direct devDependency so electron-builder runtime `require("dmg-license")` resolves under pnpm.
- Android release metadata is derived from `package.json`: `0.4.1` maps to `versionName=0.4.1`, `versionCode=401`.
