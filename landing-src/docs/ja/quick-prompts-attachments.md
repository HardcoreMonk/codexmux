タイムライン下の入力バーは単なる textarea ではありません。保存されたプロンプト、添付ファイル、メッセージ履歴が住む場所であり、1 日に 10 回打つものを 10 回打たずに済むようになります。

## クイックプロンプト

クイックプロンプトは `~/.purplemux/quick-prompts.json` に保存される短い名前付きエントリです。入力バーの上にチップとして表示され — クリック 1 回でそのプロンプトを入力したかのように送信されます。

2 つのビルトインがすぐに使え、いつでも無効化できます:

- **Commit** — `/commit-commands:commit` を実行
- **Simplify** — `/simplify` を実行

**設定 → クイックプロンプト** から自分のものを追加:

1. **プロンプトを追加** をクリック。
2. 名前 (チップのラベル) と本文 (送信内容) を指定。
3. ドラッグで並び替え。トグルで削除せずに非表示に。

本文に書いたものはそのまま送信されます — スラッシュコマンド、複数行のプロンプト、「エディタで開いているファイルを説明して 1 つの改善点を提案して」のようなテンプレート化されたリクエストも含めて。

{% call callout('tip', 'スラッシュコマンドも有効') %}
クイックプロンプトは Claude Code のスラッシュコマンドのワンクリックトリガーとして相性が良いです。「Review this PR」のチップで `/review` を呼ぶようにすれば、毎回数キーストロークを節約できます。
{% endcall %}

## 画像のドラッグ&ドロップ

画像ファイル (PNG、JPG、WebP など) を入力バーのどこかにドロップすると添付されます。purplemux はファイルをサーバ上のテンポラリパスにアップロードし、プロンプトに参照を自動で挿入します。

他にも以下が可能です:

- クリップボードから画像を直接 **ペースト**
- **クリップアイコン** をクリックしてファイルダイアログから選択
- メッセージごとに **最大 20 ファイル** を添付

添付が保留中の間は入力欄の上にサムネイルストリップが表示されます。各サムネイルには送信前に削除する X ボタンがあります。

## その他のファイル添付

同じクリップアイコンは画像以外のファイルにも使えます — markdown、JSON、CSV、ソースファイル、何でも。purplemux はそれらをテンポラリディレクトリに置き、Claude が `read` できるようパスを挿入します。

これは Claude が自力で到達できないもの — 別マシンから貼り付けたスタックトレースや別プロジェクトの設定ファイルなど — を共有する一番楽な方法です。

## モバイルフレンドリー

添付とクリップアイコンはスマートフォンでもフルサイズです。iOS の共有シートからスクリーンショットをドロップしたり、Android のカメラボタンでカメラロールから直接写真を添付したりできます。

入力バーは狭い画面に合わせてリフローします — チップは横スクロール可能なリストになり、textarea はスクロールするまで最大 5 行まで広がります。

## メッセージ履歴

ワークスペース内で送信したすべてのプロンプトはワークスペース単位の履歴に保存されます。再利用するには:

- 空の入力バーで <kbd>↑</kbd> を押して最近のメッセージを順に表示
- **履歴** ピッカーを開いて検索可能なリストを表示

古いエントリはピッカーから削除できます。履歴は他のワークスペースデータと一緒に `~/.purplemux/` に保存され、マシン外には決して送信されません。

## キーボード

| キー | アクション |
|---|---|
| <kbd>⌘I</kbd> | セッションビュー内のどこからでも入力にフォーカス |
| <kbd>Enter</kbd> | 送信 |
| <kbd>⇧Enter</kbd> | 改行を挿入 |
| <kbd>Esc</kbd> | Claude が busy の間、割り込みを送る |
| <kbd>↑</kbd> | (空のとき) メッセージ履歴を遡る |
