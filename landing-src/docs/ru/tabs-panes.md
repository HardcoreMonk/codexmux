Рабочее пространство делится на **панели**, и каждая панель содержит стек **вкладок**. Разделения дают параллельные виды; вкладки позволяют одной панели держать несколько шеллов, не съедая площадь экрана.

## Вкладки

Каждая вкладка — это настоящий шелл, привязанный к tmux-сессии. Заголовок вкладки берётся из переднеплановового процесса: введите `vim`, и вкладка переименуется; выйдите — вернётся имя каталога.

| Действие | macOS | Linux / Windows |
|---|---|---|
| Новая вкладка | <kbd>⌘T</kbd> | <kbd>Ctrl+T</kbd> |
| Закрыть вкладку | <kbd>⌘W</kbd> | <kbd>Ctrl+W</kbd> |
| Предыдущая вкладка | <kbd>⌘⇧[</kbd> | <kbd>Ctrl+Shift+[</kbd> |
| Следующая вкладка | <kbd>⌘⇧]</kbd> | <kbd>Ctrl+Shift+]</kbd> |
| Перейти на вкладку 1–9 | <kbd>⌃1</kbd> – <kbd>⌃9</kbd> | <kbd>Alt+1</kbd> – <kbd>Alt+9</kbd> |

Перетаскивайте вкладку в строке вкладок, чтобы переупорядочить. Кнопка **+** в конце строки вкладок открывает тот же выбор шаблонов, что и <kbd>⌘T</kbd>.

{% call callout('tip', 'Шаблоны не ограничиваются Terminal') %}
Меню новой вкладки позволяет выбрать тип панели: **Terminal**, **Claude**, **Diff** или **Web browser**. Все они — вкладки, их можно смешивать в одной панели и переключаться между ними теми же сокращениями.
{% endcall %}

## Разделение панелей

Вкладки делят площадь экрана. Чтобы видеть две вещи одновременно, разделите панель.

| Действие | macOS | Linux / Windows |
|---|---|---|
| Разделить вправо | <kbd>⌘D</kbd> | <kbd>Ctrl+D</kbd> |
| Разделить вниз | <kbd>⌘⇧D</kbd> | <kbd>Ctrl+Shift+D</kbd> |

Новая часть наследует каталог по умолчанию рабочего пространства и стартует с пустой вкладки терминала. У каждой панели своя строка вкладок, поэтому правая панель может показывать diff-вьюер, пока левая запускает `claude`.

## Перемещение фокуса между панелями

Используйте направленные сокращения — они обходят дерево разделений, поэтому <kbd>⌘⌥→</kbd> из глубоко вложенной панели всё равно попадёт в визуально соседнюю.

| Действие | macOS | Linux / Windows |
|---|---|---|
| Фокус влево | <kbd>⌘⌥←</kbd> | <kbd>Ctrl+Alt+←</kbd> |
| Фокус вправо | <kbd>⌘⌥→</kbd> | <kbd>Ctrl+Alt+→</kbd> |
| Фокус вверх | <kbd>⌘⌥↑</kbd> | <kbd>Ctrl+Alt+↑</kbd> |
| Фокус вниз | <kbd>⌘⌥↓</kbd> | <kbd>Ctrl+Alt+↓</kbd> |

## Изменение размера и выравнивание

Перетаскивайте разделитель между панелями для тонкой регулировки — или используйте клавиатуру.

| Действие | macOS | Linux / Windows |
|---|---|---|
| Расширить влево | <kbd>⌘⌃⇧←</kbd> | <kbd>Ctrl+Alt+Shift+←</kbd> |
| Расширить вправо | <kbd>⌘⌃⇧→</kbd> | <kbd>Ctrl+Alt+Shift+→</kbd> |
| Расширить вверх | <kbd>⌘⌃⇧↑</kbd> | <kbd>Ctrl+Alt+Shift+↑</kbd> |
| Расширить вниз | <kbd>⌘⌃⇧↓</kbd> | <kbd>Ctrl+Alt+Shift+↓</kbd> |
| Уравнять разделения | <kbd>⌘⌥=</kbd> | <kbd>Ctrl+Alt+=</kbd> |

Уравнивание — самый быстрый способ сбросить раскладку, скатившуюся в неудобные крайности.

## Очистка экрана

<kbd>⌘K</kbd> очищает терминал текущей панели — так же, как делают большинство нативных терминалов. Процесс шелла продолжает работать; стирается только видимый буфер.

| Действие | macOS | Linux / Windows |
|---|---|---|
| Очистить экран | <kbd>⌘K</kbd> | <kbd>Ctrl+K</kbd> |

## Вкладки переживают всё

Закрытие вкладки убивает её tmux-сессию. Закрытие *браузера*, обновление страницы или потеря сети — нет: каждая вкладка продолжает работать на сервере. Откройте снова, и те же панели, разделения и вкладки вернутся.

История восстановления через перезагрузку сервера — в [Сохранение и восстановление раскладок](/purplemux/ru/docs/save-restore/).
