Боковая панель и строка ввода состоят из небольших списков, форму которых можно менять — ссылки-ярлыки внизу боковой панели, кнопки промптов над вводом. Вкладка Claude в Настройках содержит CLI-уровневые переключатели для сессий, запускаемых из панели.

## Элементы боковой панели

Настройки (<kbd>⌘,</kbd>) → вкладка **Боковая панель**. Список управляет рядом ярлыков, который живёт внизу боковой панели — ссылки на дашборды, внутренние инструменты, всё, что доступно по URL.

У каждой строки есть ручка перетаскивания, имя, URL и переключатель. Можно:

- **Перетащить** ручку, чтобы переупорядочить. И встроенные, и кастомные элементы свободно двигаются.
- **Переключить** свитч, чтобы скрыть элемент без удаления.
- **Редактировать** кастомные элементы (иконка карандаша) — менять имя, иконку или URL.
- **Удалить** кастомные элементы (иконка корзины).
- **Сбросить к умолчанию** — восстанавливает встроенные элементы, удаляет все кастомные, очищает порядок.

### Добавление кастомного элемента

Нажмите **Добавить элемент** внизу. Появится небольшая форма:

- **Имя** — отображается как подсказка и подпись.
- **Иконка** — выбирается из поисковой галереи lucide-react.
- **URL** — подойдёт любой `http(s)://...`. Внутренняя Grafana, дашборды Vercel, внутренний админский инструмент.

Кликните Сохранить, и строка появится внизу списка. Перетащите её куда нужно.

{% call callout('note', 'Встроенные можно скрыть, нельзя удалить') %}
У встроенных элементов (тех, что purplemux поставляет) есть только переключатель и ручка — без редактирования и удаления. Они всегда на месте на случай, если передумаете. У кастомных полный набор.
{% endcall %}

## Быстрые промпты

Настройки → вкладка **Быстрые промпты**. Это кнопки, которые сидят над полем ввода Claude — один клик отправляет заранее написанное сообщение.

Шаблон тот же, что у элементов боковой панели:

- Перетаскивание для перестановки.
- Переключатель для скрытия.
- Редактирование / удаление кастомных промптов.
- Сбросить к умолчанию.

Добавление промпта запрашивает **имя** (подпись кнопки) и сам **промпт** (многострочный текст). Используйте для того, что часто пишете: «Запусти набор тестов», «Суммируй последний коммит», «Просмотри текущий diff».

## Опции Claude CLI

Настройки → вкладка **Claude**. Эти флаги влияют на *как purplemux запускает Claude CLI* в новых вкладках — поведение уже работающей сессии они не меняют.

### Skip Permission Checks

Добавляет `--dangerously-skip-permissions` к команде `claude`. Claude будет запускать инструменты и редактировать файлы, не спрашивая каждый раз одобрения.

Это тот же флаг, который выставляет официальный CLI — purplemux ничего не ослабляет сверху. Прочитайте [документацию Anthropic](https://docs.anthropic.com/en/docs/claude-code/cli-reference) перед включением. Включайте только для доверенных рабочих пространств.

### Show Terminal with Claude

Когда **включено** (по умолчанию): вкладка Claude показывает живой вид сессии *и* нижележащую панель терминала бок о бок, чтобы можно было заглянуть в шелл в любой момент.

Когда **выключено**: новые вкладки Claude открываются со свёрнутым терминалом. Вид сессии заполняет всю панель. Терминал по-прежнему можно развернуть вручную для каждой вкладки; настройка влияет только на дефолт для вновь создаваемых вкладок.

Используйте выключенный вариант, если вы в основном работаете с Claude через таймлайн и хотите более чистый дефолт.
