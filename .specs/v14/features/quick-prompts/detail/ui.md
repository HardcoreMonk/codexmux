# 화면 구성

## Suggestion 바 레이아웃

### 입력창 위 배치

```
┌─ Claude Code Panel ──────────────────────┐
│                                          │
│  [타임라인 영역]                           │
│                                          │
│  ┌─ Suggestion 바 ──────────────────┐    │
│  │  [ 커밋하기 ]  [ 코드 리뷰 ]      │    │
│  └────────────────────────────────────┘  │
│  ┌─ 입력창 ──────────────────────── ⏎ ┐  │
│  │ 메시지를 입력하세요...              │  │
│  └────────────────────────────────────┘  │
│                                          │
│  [터미널 영역]                            │
└──────────────────────────────────────────┘
```

- 위치: 입력창 바로 위, 타임라인과 입력창 사이
- 높이: 버튼 1줄 (~32px + padding)
- 패딩: `px-3 py-1.5`
- 배경: 투명 (입력창 영역과 일체)
- Quick Prompts가 0개이면 영역 자체 숨김 (`height: 0`)

## 버튼 스타일

### 활성 상태 (idle)

```
┌─────────────┐
│  커밋하기    │
└─────────────┘
```

- `variant="outline" size="sm"`
- `text-xs`, `border-dashed`, `rounded-full`
- `px-3 py-1`
- `text-muted-foreground hover:text-foreground hover:border-foreground/30`
- 버튼 간 간격: `gap-2`
- 클릭 가능: `cursor-pointer`

### 비활성 상태 (busy / inactive)

- `opacity-50`, `pointer-events-none`, `cursor-default`
- 호버 효과 없음
- 시각적으로 흐릿하게 — 클릭 불가임을 암시

## 모바일

```
┌─────────────────────────────┐
│  ☰  project / claude   [⌨]  │
│─────────────────────────────│
│  [타임라인]                   │
│─────────────────────────────│
│  [ 커밋하기 ] [ 코드 리뷰 ]   │  ← suggestion 바
│─────────────────────────────│
│  메시지를 입력하세요...   ⏎  │
└─────────────────────────────┘
```

- 데스크톱과 동일 레이아웃
- 버튼이 많으면 가로 스크롤 (`overflow-x-auto`, 스크롤바 숨김)
- 터치 타겟: 최소 32px 높이

## 설정 UI

### Quick Prompts 설정 섹션

```
┌─ 설정 ──────────────────────────────────┐
│                                         │
│  빠른 프롬프트                           │
│                                         │
│  ┌─────────────────────────────────────┐│
│  │ 커밋하기                        [✓] ││
│  │ /commit-commands:commit             ││
│  │                          [수정] [삭제]││
│  ├─────────────────────────────────────┤│
│  │ 코드 리뷰                       [✓] ││
│  │ 현재 변경사항을 리뷰해주세요.        ││
│  │                          [수정] [삭제]││
│  └─────────────────────────────────────┘│
│                                         │
│  [ + 프롬프트 추가 ]                     │
│                                         │
│  [ 기본값으로 초기화 ]                    │
│                                         │
└─────────────────────────────────────────┘
```

### 항목 구성

| 요소 | 스타일 |
|---|---|
| 이름 | `text-sm font-medium` |
| prompt | `text-xs text-muted-foreground font-mono truncate` |
| 토글 | shadcn/ui `Switch` |
| 수정 | `variant="ghost" size="sm"` (lucide `Pencil`) |
| 삭제 | `variant="ghost" size="sm"` (lucide `Trash2`, `text-ui-red`) |

### 추가/수정 폼

```
┌─ 프롬프트 추가 ─────────────────────────┐
│                                         │
│  이름                                   │
│  ┌─────────────────────────────────────┐│
│  │                                     ││
│  └─────────────────────────────────────┘│
│                                         │
│  프롬프트                               │
│  ┌─────────────────────────────────────┐│
│  │                                     ││
│  │                                     ││
│  └─────────────────────────────────────┘│
│                                         │
│              [ 취소 ]  [ 저장 ]          │
└─────────────────────────────────────────┘
```

- 인라인 폼 또는 다이얼로그
- 이름: shadcn/ui `Input`, 필수
- 프롬프트: shadcn/ui `Textarea`, 필수, 여러 줄 가능
- 저장: `variant="default"`, 취소: `variant="outline"`

## 반응형/접근성

- 버튼: `tabIndex={0}`, Enter/Space로 클릭
- suggestion 바: `role="toolbar"`, 화살표 키로 버튼 간 이동
- 설정 토글: 키보드 접근 가능
