# 화면 구성

## 전체 레이아웃

### Claude Code Panel (입력창 포함)

```
┌─ PaneContainer ──────────────────────────────────┐
│  ┌─ 탭 바 ───────────────────────────────────┐   │
│  │ [Tab1] [Tab2]          [⬚] [─] [│] [✕]    │   │
│  └───────────────────────────────────────────┘   │
│  ┌─ Claude Code Panel ──────────────────────┐    │
│  │  ┌─ 타임라인 영역 ────────────────────┐  │    │
│  │  │                                    │  │    │
│  │  │  (기존 Phase 8 타임라인)            │  │    │
│  │  │                                    │  │    │
│  │  └────────────────────────────────────┘  │    │
│  │  ━━━━━━━━━━━ 리사이즈 핸들 ━━━━━━━━━━━━  │    │
│  │  ┌─ 입력창 (고정, 오버레이) ─────── ⏎ ┐  │    │
│  │  │ 메시지를 입력하세요...             │  │    │
│  │  └────────────────────────────────────┘  │    │
│  │  ┌─ 터미널 영역 (축소) ───────────────┐  │    │
│  │  │  $ claude --resume ...             │  │    │
│  │  └────────────────────────────────────┘  │    │
│  └──────────────────────────────────────────┘    │
└──────────────────────────────────────────────────┘
```

- 리사이즈 핸들은 타임라인 ↔ (입력창 + 터미널) 사이
- 입력창은 `react-resizable-panels` 외부, 터미널 영역 상단에 고정
- 입력창이 확장되면 터미널 영역 위로 **오버레이** (z-index로 겹침)

## 입력창 컴포넌트

### 입력 모드 (CLI 입력 대기)

```
┌────────────────────────────────────────────────┐
│  메시지를 입력하세요...                      ⏎  │
└────────────────────────────────────────────────┘
```

- 전체: `relative px-3 py-2`, `bg-background`, `border-t` (0.5px)
- textarea: `flex-1 resize-none bg-transparent text-sm text-foreground outline-none`
  - placeholder: `text-muted-foreground`
  - 기본 높이: 1줄 (~20px + padding)
  - autosize: 내용에 따라 최대 5줄까지 확장
- Send 버튼: 우측, `SendHorizontal` 아이콘 (`size={16}`)
  - `variant="ghost" size="sm"`, `text-muted-foreground hover:text-foreground`
  - 입력이 있을 때만 `text-ui-purple` 활성 색상
- 포커스 시: `border-t` 색상 → `border-ring` (accent)

### 여러 줄 입력 상태 (오버레이 확장)

```
┌────────────────────────────────────────────────┐
│  이 함수를 리팩토링해줘.                         │
│  특히 에러 처리 부분을 개선하고,                   │
│  타입 안전성을 높여줘.                       ⏎  │
└────────────────────────────────────────────────┘
┌─ 터미널 영역 (일부 가려짐) ────────────────────┐
│  $ claude --resume ...                        │
└────────────────────────────────────────────────┘
```

- 입력창이 아래로 확장 → `position: relative` + `z-10`으로 터미널 위에 겹침
- 터미널 영역은 크기 변경 없음, 시각적으로 일부 가려짐
- 최대 높이: 5줄 (~100px + padding), 이후 내부 스크롤

### 중단 모드 (CLI 처리 중)

```
┌────────────────────────────────────────────────┐
│  Claude가 응답 중...                         ■  │
└────────────────────────────────────────────────┘
```

- textarea: `disabled`, `opacity-70`, `cursor-not-allowed`
- placeholder: "Claude가 응답 중..." (`text-muted-foreground`)
- Send 버튼 → 중단 버튼으로 교체:
  - 아이콘: lucide-react `Square` (`size={14}`, `fill="currentColor"`)
  - 색상: `text-ui-red hover:text-ui-red/80`
  - 클릭 → AlertDialog 표시

### 비활성 모드 (CLI 종료)

```
┌────────────────────────────────────────────────┐
│  Claude Code가 실행 중이 아닙니다                │
└────────────────────────────────────────────────┘
```

- 전체: `opacity-50`
- textarea: `disabled`, `cursor-not-allowed`
- Send 버튼: `disabled`, `opacity-30`
- placeholder: "Claude Code가 실행 중이 아닙니다"

### 숨김 상태 (세션 목록 뷰 / Terminal 모드)

- 입력창 영역 높이 → 0, `overflow-hidden`, `transition: height 150ms ease-out`
- 표시 시: 0 → auto (내용 높이)

## 중단 확인 다이얼로그

```
┌─────────────────────────────────────────┐
│  작업 중단                               │
│                                         │
│  Claude 작업을 중단하시겠습니까?          │
│  진행 중인 작업이 취소됩니다.             │
│                                         │
│              [ 취소 ]  [ 중단 ]          │
└─────────────────────────────────────────┘
```

- shadcn/ui `AlertDialog`
- 타이틀: "작업 중단"
- 설명: "Claude 작업을 중단하시겠습니까? 진행 중인 작업이 취소됩니다."
- 취소 버튼: `variant="outline"`
- 중단 버튼: `variant="destructive"`

## 반응형/접근성

- textarea: `aria-label="Claude Code 메시지 입력"`
- Send 버튼: `aria-label="메시지 전송"` / 중단 버튼: `aria-label="작업 중단"`
- **Cmd/Ctrl+I**: 입력창 포커스
- **Escape**: 입력창에서 터미널로 포커스 이동
- **Enter**: 전송
- **Shift+Enter**: 줄바꿈
- **Tab**: 입력창 내에서 포커스 트랩하지 않음 (자연스러운 탭 이동)
