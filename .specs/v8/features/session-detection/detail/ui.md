# 화면 구성

## 개요

세션 감지 및 매핑은 서버 측 모듈이므로 직접적인 UI 요소가 없다. 감지 결과에 따른 클라이언트 상태 변화를 정의한다.

## 감지 상태에 따른 UI 반영

### 활성 세션 감지됨

```
┌─ Claude Code Panel ──────────────────────┐
│  ● 활성 세션 연결됨                        │
│  ┌─ 타임라인 ─────────────────────────┐  │
│  │ (세션 파일 파싱 → 타임라인 표시)     │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- 타임라인 WebSocket 연결 → 실시간 업데이트 시작
- 세션 상태 표시 불필요 (타임라인이 있으면 연결된 상태)

### 활성 세션 없음 → 최근 세션 표시

```
┌─ Claude Code Panel ──────────────────────┐
│  ┌─ 타임라인 ─────────────────────────┐  │
│  │ (가장 최근 세션 파일 표시)           │  │
│  │ (실시간 업데이트 없음 — 정적)       │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- 최근 수정된 JSONL 파일의 내용을 정적 표시
- 새 세션 시작 시 자동 전환

### 세션 파일 없음

```
┌─ Claude Code Panel ──────────────────────┐
│                                          │
│     ┌─────────────────────────────┐      │
│     │  ○ Claude Code 세션 없음     │      │
│     │                             │      │
│     │  이 프로젝트에서 Claude Code  │      │
│     │  세션이 아직 없습니다.        │      │
│     └─────────────────────────────┘      │
│                                          │
└──────────────────────────────────────────┘
```

- 빈 상태 UI: 중앙 정렬 메시지
- `text-muted-foreground` 색상
- Claude Code 실행 시 자동 전환

### Claude Code 미설치

```
┌─ Claude Code Panel ──────────────────────┐
│                                          │
│     ┌─────────────────────────────┐      │
│     │  ○ Claude Code를 설치하세요  │      │
│     │                             │      │
│     │  ~/.claude 디렉토리를        │      │
│     │  찾을 수 없습니다.           │      │
│     └─────────────────────────────┘      │
│                                          │
└──────────────────────────────────────────┘
```

## 세션 전환 시 UI 변화

```
기존 세션 타임라인 표시 중
├── 새 세션 감지됨 (session-changed)
├── 타임라인 초기화 (기존 엔트리 제거)
├── 스켈레톤 로딩 표시 (200ms 이내)
└── 새 세션 데이터 로드 → 타임라인 렌더링
```
