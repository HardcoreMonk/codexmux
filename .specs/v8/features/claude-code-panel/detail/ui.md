# 화면 구성

## 개요

Claude Code Panel은 상단 타임라인 + 하단 축소 터미널로 구성된 분할 레이아웃이다. 타임라인 영역이 대부분의 공간을 차지하고, 터미널은 축소 표시된다.

## 전체 레이아웃

```
┌─ PaneContainer ──────────────────────────────────┐
│  ┌─ 탭 바 ───────────────────────────────────┐   │
│  │ [Tab1] [Tab2]          [⬚] [─] [│] [✕]    │   │
│  └───────────────────────────────────────────┘   │
│  ┌─ Claude Code Panel ──────────────────────┐    │
│  │  ┌─ 타임라인 영역 ────────────────────┐  │    │
│  │  │                                    │  │    │
│  │  │  (스크롤 가능, 대부분의 공간)        │  │    │
│  │  │                                    │  │    │
│  │  └────────────────────────────────────┘  │    │
│  │  ━━━━━━━━━━━ 리사이즈 핸들 ━━━━━━━━━━━━  │    │
│  │  ┌─ 터미널 영역 (축소 50%) ───────────┐  │    │
│  │  │  $ claude                          │  │    │
│  │  │  > ...                             │  │    │
│  │  └────────────────────────────────────┘  │    │
│  └──────────────────────────────────────────┘    │
└──────────────────────────────────────────────────┘
```

## 타임라인 엔트리 UI

### 사용자 메시지

```
┌──────────────────────────────────────────┐
│  14:30                                   │
│  ┌─────────────────────────────────────┐ │
│  │ 버그 수정해줘                        │ │
│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

- 타임스탬프: `text-xs text-muted-foreground`
- 메시지 블록: `bg-ui-blue/10` 배경, `border-l-2 border-ui-blue` 좌측 액센트
- 패딩: `px-4 py-3`

### 어시스턴트 응답

```
┌──────────────────────────────────────────┐
│  14:32                                   │
│  ┌─────────────────────────────────────┐ │
│  │ 수정 완료했습니다. `src/main.ts`에서 │ │
│  │ 발생하던 null 참조 오류를 수정...    │ │
│  │                                     │ │
│  │ ```typescript                       │ │
│  │ const value = data?.name ?? '';     │ │
│  │ ```                                 │ │
│  │                              더 보기 │ │
│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

- 마크다운 렌더링: `react-markdown` + 코드 블록 구문 강조
- 메시지 블록: `bg-ui-purple/5` 배경, `border-l-2 border-ui-purple` 좌측 액센트
- 긴 응답 (10줄 이상): 초기 접기 + "더 보기" 버튼
- "더 보기" 버튼: `text-xs text-muted-foreground hover:text-foreground`

### 도구 호출

```
┌──────────────────────────────────────────┐
│  14:31                                   │
│  ┌─ ✅ ─────────────────────────────────┐ │
│  │ ✎ Edit src/main.ts (+3, -1)         │ │
│  │ ▸ diff 보기                          │ │
│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

- 아이콘: lucide-react (도구별 매핑, `size={14}`)
- 상태 인디케이터:
  - 성공: `text-positive` (ui-teal)
  - 실패: `text-negative` (ui-red)
  - 진행 중: `text-ui-amber` + `animate-pulse`
- 요약 텍스트: `text-sm font-mono`
- diff 토글: `text-xs text-muted-foreground cursor-pointer`
- 전체 블록: `bg-muted/50` 배경, `rounded-md`, `px-3 py-2`

### diff 뷰 (펼침)

```
┌──────────────────────────────────────────┐
│  14:31                                   │
│  ┌─ ✅ ─────────────────────────────────┐ │
│  │ ✎ Edit src/main.ts (+3, -1)         │ │
│  │ ▾ diff 숨기기                        │ │
│  │ ┌───────────────────────────────────┐│ │
│  │ │ - const value = data.name;       ││ │  ← bg-ui-red/10
│  │ │ + const value = data?.name ?? '';││ │  ← bg-ui-teal/10
│  │ └───────────────────────────────────┘│ │
│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

- diff 컨테이너: `bg-background rounded border` (0.5px)
- 삭제 줄: `bg-ui-red/10` 배경, `text-ui-red` 접두사 `-`
- 추가 줄: `bg-ui-teal/10` 배경, `text-ui-teal` 접두사 `+`
- 폰트: `font-mono text-xs`

### 서브에이전트 그룹 (접힌 상태)

```
┌──────────────────────────────────────────┐
│  14:31                                   │
│  ┌─────────────────────────────────────┐ │
│  │ ▸ Agent: Explore — 코드베이스 탐색   │ │
│  │   (12 단계 완료)                     │ │
│  └─────────────────────────────────────┘ │
└──────────────────────────────────────────┘
```

- 전체 블록: `bg-muted/30` 배경, `text-muted-foreground`
- 접힌 화살표: `▸` (v8에서는 펼치기 불가)
- 에이전트 타입/설명: `text-sm`
- 단계 수: `text-xs text-muted-foreground`

## 자동 스크롤

### "최신으로 이동" 버튼

```
┌─ 타임라인 영역 ──────────────────────────┐
│  (사용자가 위로 스크롤하여 과거 내용 조회) │
│  ...                                     │
│                                          │
│              ┌─────────────────┐         │
│              │ ↓ 최신으로 이동  │         │  ← 플로팅 버튼
│              └─────────────────┘         │
└──────────────────────────────────────────┘
```

- 위치: 타임라인 영역 하단 중앙, 플로팅
- 스타일: `bg-background border shadow-sm rounded-full px-3 py-1 text-xs`
- 클릭 → 부드러운 스크롤 → 자동 스크롤 재개
- 자동 스크롤 활성 시 숨김

## 터미널 축소 영역

```
┌─ 터미널 영역 (축소) ─────────────────────┐
│  ┌──────────────────────────────────────┐│
│  │ $ claude                             ││  ← CSS transform: scale(0.5)
│  │ ╭─────────────────────────────────╮  ││     origin: top left
│  │ │ > 입력 대기                      │  ││     80 컬럼 기준
│  │ ╰─────────────────────────────────╯  ││
│  └──────────────────────────────────────┘│
└──────────────────────────────────────────┘
```

- xterm.js 인스턴스: 80 컬럼 기준 cols/rows 설정
- CSS `transform: scale(0.5)`, `transform-origin: top left`
- 컨테이너 크기: 축소 비율에 맞게 역산 (실제 터미널 크기의 50%)
- 포커스 가능: 클릭 시 xterm.js `focus()` 호출

## 빈 상태

### 세션 없음

```
┌─ 타임라인 영역 ──────────────────────────┐
│                                          │
│         ○ Claude Code 세션 없음           │
│                                          │
│     이 프로젝트에서 Claude Code            │
│     세션이 아직 없습니다.                   │
│                                          │
└──────────────────────────────────────────┘
```

- 중앙 정렬, `text-muted-foreground`
- 아이콘: lucide-react `MessageSquare` (muted)

### 로딩 중

```
┌─ 타임라인 영역 ──────────────────────────┐
│  ┌─ 스켈레톤 ─────────────────────────┐  │
│  │ ░░░░░░░░░░░░░░░░░░░░              │  │
│  │ ░░░░░░░░░░░░                      │  │
│  │ ░░░░░░░░░░░░░░░░░░                │  │
│  └────────────────────────────────────┘  │
└──────────────────────────────────────────┘
```

- 3~5개의 스켈레톤 블록, 높이 다양
- `animate-pulse`, `bg-muted` 배경, `rounded-md`

## 반응형/접근성

- 타임라인 영역: 키보드 스크롤 (`ArrowUp/Down`, `PageUp/Down`, `Home/End`)
- diff 토글: Enter/Space로 접기/펼치기
- "더 보기" 버튼: Enter/Space로 확장
- 타임라인-터미널 리사이즈: `react-resizable-panels` 기본 키보드 지원
