# 화면 구성

## 전체 레이아웃

### 단일 Pane (Phase 3 호환)

```
┌─ Tab 1 ──┬─ Tab 2* ─┬─ + ──┬─ ┃ ━ ──┐  ← Pane 탭 바 (30px)
├────────────────────────────────────────┤
│                                        │
│           xterm.js 터미널               │
│           (활성 탭의 세션)               │
│                                        │
│                                [상태]  │
└────────────────────────────────────────┘
```

### 수직 2분할

```
┌─ Tab A1* ─┬─ + ─┬─ ┃ ━ ─┐│┌─ Tab B1* ─┬─ + ─┬─ ┃ ━ ─┬─ × ─┐
├────────────────────────────┤│├─────────────────────────────────┤
│                            ││                                   │
│   Pane A (터미널)           ▐▐   Pane B (터미널)                 │
│   [포커스 보더]             ││                                   │
│                            ││                                   │
└────────────────────────────┘│└─────────────────────────────────┘
                              ↑
                         Separator (드래그 리사이즈)
```

### 3분할 (수직 + 수평)

```
┌─ Tab A1* ─┬─ + ─┬─ ━ ──┬─ × ─┐│┌─ Tab B1* ─┬─ + ─┬─ × ──┐
├────────────────────────────────┤│├────────────────────────────┤
│                                ││                              │
│   Pane A (터미널)               ││   Pane B (터미널)             │
│                                ││                              │
├─ Tab C1* ─┬─ + ─┬─ × ────────┤│                              │
├────────────────────────────────┤│                              │
│                                ││                              │
│   Pane C (터미널)               ││                              │
│                                ││                              │
└────────────────────────────────┘└──────────────────────────────┘
```

- Pane 3개 도달: 분할 버튼(┃ ━) 비활성화 (dimmed)
- 포커스된 Pane에만 `ui-purple` 보더 표시

## 컴포넌트 구조

```
pages/index.tsx
└── <TerminalPage />
    └── <PaneLayout />                         ← 신규: 트리 렌더러
        ├── <Group orientation="horizontal">    ← react-resizable-panels v4
        │   ├── <Panel minSize={200}>
        │   │   └── <PaneContainer />           ← 신규: 개별 Pane
        │   │       ├── <PaneTabBar />          ← tab-bar.tsx 확장
        │   │       │   ├── <Tab /> (반복)
        │   │       │   ├── <AddTabButton />
        │   │       │   ├── <SplitButtons />    ← 신규: ┃ ━ 버튼
        │   │       │   └── <PaneCloseButton /> ← 신규: × (Pane 닫기)
        │   │       ├── <TerminalContainer />   ← Phase 2 재활용
        │   │       │   └── xterm.js (Pane별 독립)
        │   │       └── <ConnectionStatus />
        │   ├── <Separator />                   ← 분할선
        │   └── <Panel minSize={200}>
        │       └── <PaneContainer /> ...
        └── (재귀 중첩 가능)
```

## Pane 탭 바 (`PaneTabBar`)

Phase 3 `TabBar`를 확장하여 Pane별 독립 동작 + 분할/닫기 버튼을 추가한다.

| 속성 | 값 | 설명 |
|---|---|---|
| 높이 | 30px | Phase 3과 동일 |
| 배경 | `oklch(0.18 0.006 286)` | Zinc 다크 |
| 하단 보더 | `0.5px` Zinc 600 수준 | Pane 탭 바와 터미널 경계 |
| 좌측 | 탭 목록 (스크롤 가능) | Phase 3 동일 |
| 우측 | `+` → `┃` → `━` → `×` | 고정 영역 |

### 분할 버튼 (`SplitButtons`)

| 속성 | 값 |
|---|---|
| 아이콘 | lucide `PanelRight` (수직 ┃), `PanelBottom` (수평 ━) |
| 크기 | 24×24px, 아이콘 12px |
| hover | 배경 `oklch(0.24 0.006 286)`, 원형 |
| 비활성 (3개 도달 또는 공간 부족) | `opacity: 0.3`, 클릭 불가, `cursor: not-allowed` |
| 툴팁 | "수직 분할" / "수평 분할" |

### Pane 닫기 버튼 (`PaneCloseButton`)

| 속성 | 값 |
|---|---|
| 아이콘 | lucide `X` (12px) |
| 크기 | 24×24px |
| 표시 조건 | Pane이 2개 이상일 때만 |
| hover | 배경 `oklch(0.28 0.02 20)` (ui-red 계열 연한 톤) |
| 툴팁 | "Pane 닫기" |

## 포커스 보더

| 상태 | 보더 | 설명 |
|---|---|---|
| 포커스 | `1px solid` `ui-purple` (`oklch(0.71 0.051 289)`) | Pane 전체 래퍼에 적용 |
| 비포커스 | 없음 | 보더 제거 (패딩으로 1px 보정하여 레이아웃 시프트 방지) |

- Pane이 1개일 때: 포커스 보더 미표시 (불필요)
- 포커스 전환 시 보더 150ms transition

## 분할선 (`Separator`)

| 속성 | 값 |
|---|---|
| 시각적 두께 | 1px |
| 색상 (기본) | `oklch(0.30 0.006 286)` (Zinc 계열, 15% 불투명도) |
| 색상 (hover) | `oklch(0.40 0.006 286)` (약간 밝게) |
| 색상 (드래그 중) | `oklch(0.50 0.010 286)` |
| cursor | `col-resize` (수직) / `row-resize` (수평) |
| 히트 영역 | 라이브러리 자동 처리 (시각보다 넓음) |
| transition | `background-color 100ms` |

## 탭 드래그 앤 드롭 (Pane 간 이동)

### 드래그 중

```
          ↓ 커서
┌─ Tab A1 ─┬─ [Tab A2] ─┐    ← 반투명 고스트 (opacity: 0.6, scale: 0.9)
│                        │
```

- 드래그 원본 탭: `opacity: 0.3` (자리 표시)
- 고스트: 탭의 축소 복제본, 커서 우하단에 표시

### 드롭 대상 표시

```
┌─ Tab B1 ─┤│├─ Tab B2 ─┬─ + ─┐   ← 삽입 인디케이터 (2px ui-blue)
│ [탭 바 하이라이트]                │   ← 탭 바 전체 배경 변화
```

| 요소 | 스타일 |
|---|---|
| 타겟 Pane 탭 바 하이라이트 | 배경 `ui-blue/10` (10% 불투명도) |
| 삽입 인디케이터 | 2px `ui-blue` 세로 라인, 탭 사이에 표시 |
| 드롭 불가 영역 | 같은 Pane 내 → 순서 변경 (Phase 3 동작), 터미널 영역 → 무시 |

## 상태별 UI

### 로딩 (레이아웃 조회 중)

```
┌──────────────────────────────────────────┐
│  ▓▓▓▓▓▓  ▓▓▓▓▓▓  +  ┃ ━                  │  ← 탭 스켈레톤
├──────────────────────────────────────────┤
│                                          │
│              연결 중...                    │
│                                          │
└──────────────────────────────────────────┘
```

- Phase 3과 동일한 스켈레톤, 단일 Pane 구조로 표시
- 레이아웃 로드 완료 후 실제 트리로 교체

### 에러 (레이아웃 조회 실패)

```
┌──────────────────────────────────────────┐
│  ⚠ 레이아웃을 불러올 수 없습니다.  [재시도]  │
├──────────────────────────────────────────┤
│                                          │
└──────────────────────────────────────────┘
```

- 재시도 클릭 → 레이아웃 재조회
- 반복 실패 시 기본 단일 Pane으로 폴백 + toast ("기본 레이아웃으로 시작합니다")

### 빈 상태

- 발생하지 않음 — 레이아웃이 없으면 기본 단일 Pane 자동 생성

### 분할 중

```
포커스된 Pane에서 ┃ 클릭
→ 분할 버튼 disabled (로딩 피드백)
→ 서버에 새 Pane 생성 요청
→ 트리 갱신 → Separator가 나타나며 새 Pane 0→50% 확장 (200ms ease-out)
→ 새 Pane에 터미널 렌더링
→ 분할 버튼 재활성화
```

## 접근성

- 각 Pane: `role="region"`, `aria-label="Pane {N}"`
- Pane 내 탭 바: `role="tablist"`, Phase 3 접근성 속성 유지
- `Separator`: 라이브러리가 `role="separator"`, `aria-orientation`, 키보드(화살표) 자동 제공
- 포커스된 Pane: `aria-current="true"`
- 분할 버튼: `aria-label="수직 분할"` / `aria-label="수평 분할"`
- Pane 닫기: `aria-label="Pane 닫기"`
- 비활성 분할 버튼: `aria-disabled="true"`
