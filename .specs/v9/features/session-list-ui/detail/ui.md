# 화면 구성

## 개요

Claude Code Panel 상단 영역에서 세션 목록 뷰, 빈 상태 뷰, 타임라인 뷰(기존)를 상태에 따라 전환한다.

## 전체 레이아웃

## 세션 목록 헤더

```
┌──────────────────────────────────────────┐
│  Claude Code 세션                    ↻   │
└──────────────────────────────────────────┘
```

- 좌측: 타이틀 `text-sm font-medium`
- 우측: 새로고침 버튼 (lucide-react `RefreshCw`, `size={14}`)
  - 스타일: `variant="ghost" size="sm"`, `text-muted-foreground`
  - 클릭 시: 아이콘 `animate-spin` (1회전 후 중단)
- 하단 구분선: `border-b` (0.5px)
- 패딩: `px-4 py-2`

## 세션 항목

```
┌──────────────────────────────────────────┐
│  ● 03/21 14:30                2시간 전   │
│    "버그 수정해줘"                  12턴   │
└──────────────────────────────────────────┘
```

- 전체: `px-4 py-3 cursor-pointer`
- 호버: `bg-muted` 배경
- 1행 좌측: 절대 시간 `text-xs text-muted-foreground`
- 1행 우측: 상대 시간 `text-xs text-muted-foreground`
  - Tooltip: 절대 시간 전체 (`YYYY-MM-DD HH:mm:ss`)
- 2행 좌측: 첫 메시지 `text-sm truncate` (1줄)
  - Tooltip: 전체 메시지
- 2행 우측: 턴 수 배지 `text-xs text-muted-foreground`
- 항목 간 구분: `border-b` (0.5px, `border-border/50`)

### 하이라이트 세션 (현재 Surface의 claudeSessionId와 일치)

- 좌측 `●` 마커: `text-ui-purple` 인라인 원형 아이콘 (`size={6}`)
- 배경: `bg-ui-purple/5`
- 시간 텍스트: `text-ui-purple` (일반 세션은 `text-muted-foreground`)

### resume 진행 중 세션

- 클릭한 항목에 로딩 스피너: lucide-react `Loader2` + `animate-spin`
- 나머지 항목: `opacity-50 pointer-events-none`

## 타임라인 네비게이션 바

```
┌──────────────────────────────────────────┐
│  ← 세션 목록                              │
└──────────────────────────────────────────┘
```

- 좌측: `ChevronLeft` 아이콘 (`size={16}`) + "세션 목록" 텍스트
- 스타일: `variant="ghost"`, `text-sm text-muted-foreground hover:text-foreground`
- 패딩: `px-4 py-2`
- 하단 구분선: `border-b` (0.5px)

## 빈 상태 뷰

```
┌─ 세션 목록 영역 ──────────────────────────┐
│                                          │
│            ○ 세션 없음                    │
│                                          │
│     터미널에서 claude를 실행하여           │
│     새 세션을 시작하세요                   │
│                                          │
└──────────────────────────────────────────┘
```

- 중앙 정렬
- 아이콘: lucide-react `MessageSquare` (`size={32}`, `text-muted-foreground/50`)
- 타이틀: `text-sm font-medium text-muted-foreground`
- 설명: `text-xs text-muted-foreground/70`

```

│  ┌─ 스켈레톤 ─────────────────────────┐  │

│  │ ░░░░░░░░░░░░░░░░            ░░    │  │
│  ├────────────────────────────────────┤  │
│  │ ░░░░░░░░░░░           ░░░░░░      │  │
│  │ ░░░░░░░░░░░░░░                ░░  │  │
```

### 추가 로드 (스크롤 페이지네이션)

- 목록 하단: `Loader2` 스피너 (`animate-spin`, `size={16}`, `text-muted-foreground`)
- 중앙 정렬, `py-3`

## 에러 상태

```
┌─ 세션 목록 영역 ──────────────────────────┐
│                                          │
│     세션 목록을 불러올 수 없습니다         │
│                                          │
│              [ 다시 시도 ]                │
│                                          │
└──────────────────────────────────────────┘
```

- 메시지: `text-sm text-muted-foreground`
- 버튼: `variant="outline" size="sm"`
- 중앙 정렬

## 반응형/접근성

- 세션 목록: 키보드 탐색 (`ArrowUp/Down`으로 항목 이동, `Enter`로 선택)
- `← 세션 목록` 버튼: 키보드 포커스 가능, `Enter/Space`로 동작
- 새로고침 버튼: 키보드 접근 가능
- 스크롤 영역: `overflow-y-auto`, 마우스 휠 + 키보드 스크롤
